☁ Supabase 동기화 설정

🖥 PC와 모바일에서 동일한 데이터를 사용하려면 설정이 필요합니다.
• Supabase URL과 anon KEY를 입력하면 모든 기기에서 자동 동기화됩니다.
• 데이터 변경 시 자동 업로드, 페이지 로드 시 자동 다운로드.
• 테이블: kv_store (key TEXT PK, value JSONB)
• 저장되는 키: 4m_iflow_v1
▶ kv_store 테이블이 없을 때 실행할 SQL 
create table if not exists kv_store (
  key text primary key,
  value jsonb,
  updated_at timestamptz default now()
);
alter table kv_store enable row level security;
create policy "anon_all" on kv_store
  for all using (true) with check (true);
